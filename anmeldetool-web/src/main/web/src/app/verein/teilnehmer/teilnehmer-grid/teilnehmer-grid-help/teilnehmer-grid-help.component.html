1. Eingabe und ändern von Teilnehmer

Teilnehmer können in der Tabelle eingesehen und bearbeitet werden. Das Bearbeiten eines Feldes geschieht durch DoppelKlick auf dem Feld.

2. Zurücksetzen der aktuellen Kategorie

Falls ungewollt eine zu tiefe Kategorie eingegeben wurde, kann dies nur durch überschreiben des Feldes "aktuelle Kategorie" rückgängig gemacht werden.

Dazu muss in diesem Feld die tiefere Kategorie eingtragen werden. Danach kann nach einem wechsel auf "Alle Anlässe" und zurück die richtige Kategorie gesetzt werden.

3. Teilnahme an Wettkämpfen

Einzelne Starts/Teilnahmen können nur erfasst werden, wenn dein Verein am Wettkampf startet. Unter der Anlass Übersicht "Verein startet" anwählen, danach können einzelne Starts erfasst werden

4. Teilnehmer Grid

Im Teilnehmer Grid können sowohl die Angaben der Teilnehmer als auch die Starts an Wettkämpfen gepflegt werden. Mit den Knöpfen "Turnerinnen" oder "Turner" kann ein Filter auf das entsprechende Geschlecht gesetzt werden. Klicken auf den Spaltenkopf "Name" sortiert die Tabelle nach Namen. Klicken auf den Spaltenkopf "Kategorie" sortiert die Tabelle nach Kategorie.

5. Teilnehmer Grid Daten Filtern

Um die angezeigten Daten zu filtern, kann in der Kopfzeile der Tabelle ein Filter definiert werden. Durch Klicken auf die 3 horizontalen Striche rechts neben dem Spaltenkopf kann ein Filter definiert werden.
